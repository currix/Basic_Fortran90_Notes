Fortran 90 Lessons for Computational Chemistry

Abstract

The present document is a basic introduction to the Fortran programming language based in several textbooks and references (see Referencias, Chapter 13). It contains the basic scheme of Fortran programming taught in the Computational Chemistry module (fourth year, second semester) of the University of Huelva Chemistry Degree.
